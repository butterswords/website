# Books to Organize

This is a temporary staging doc, not published content (it lives under
`docs/`, outside `content/`, so Hugo won't render it). It's everything
that used to live below "Nonfiction" on `/bookshelf/`, pulled out so
the page itself could be rewritten as an intro + filterable grid.

Each entry below should eventually become its own page:

```
hugo new --kind book_review_template content/bookshelf/book-slug.md
```

...with `category` set to one of: Formative, Inspirations, In Progress,
Backlog, Recommendations, Reviews. The old section headings are kept
here only as a starting hint for that mapping — they don't line up
1:1 with the new categories, so don't assume "Critical Reads" ==
"Reviews" without thinking about each book individually.

Delete this file once everything below has been migrated.
